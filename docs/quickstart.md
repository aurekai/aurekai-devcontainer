# Quickstart — aurekai-devcontainer

Dev Container definition for Aurekai.

## Open in VS Code

1. Clone this repo
2. Open in VS Code → "Reopen in Container"

## What's included

- Python 3.11
- `akai` CLI at `0.8.0-alpha.4`
- Python, YAML, JSON extensions

## Validate

```bash
bash tests/validate-schemas.sh
bash tests/validate-scripts.sh
```
